What is VBA

Visual Basic for Applications is a programming language used for Excel and other Microsoft Office applications. Its syntax is similar to python, however unlike python, VBA can only be run within a host application.

History of VBA

How VBA works

VBA programs are executed in macros which are automated tasks/actions that are programmable. There are two ways to make a macro.

  • Recording a macro
  • Writing a macro

Why VBA?

VBA is widely used since it is easy to use and is compatible with programs that don't require a computer science to understand.

Writing a macro to change the background color of cells in column 1

    Sub sbPrintColorIndexColors()
    Dim iCntr
    For iCntr = 1 To 56
        Cells(iCntr, 1).Interior.ColorIndex = iCntr
        Cells(iCntr, 1) = iCntr
    Next iCntr
    End Sub

Instructions

  1. Open an excel workbook
  2. Press Alt+F11 to open VBA Editor
  3. Insert a new module from Insert menu
  4. Copy the above code and Paste in the code window
  5. Save the file as macro enabled workbook
  6. Press F5 to execute the procedure
  7. You can see the interior colors are changing as per our code

Writing a macro to insert multiple rows

    Sub InsertMultipleRows()
    Dim i As Integer
    Dim j As Integer
    ActiveCell.EntireRow.Select
    On Error GoTo Last
    i = InputBox("Enter number of columns to insert", "Insert Columns")
    For j = 1 To i
    Selection.Insert Shift:=xlToDown, CopyOrigin:=xlFormatFromRightorAbove
    Next j
    Last: Exit Sub
    End Sub

How to record a macro

Instructions

  1. On the View tab, in the Macros group, click the Macros button and then choose Record Macro
  2. In the Record Macro dialog box, enter a name (such as Macro1) in the Macro Name box
  3. Perform whatever tasks you would like the macro to perform
  4. Stop recording once done
  5. Run the macro as you would with any other macro
